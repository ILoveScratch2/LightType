# LightType

这是一个简单的，基于网页的 Markdown 编辑器，单文件可用，不依赖任何第三方库

> 语言：**HTML CSS JavaScript**

## 支持列表

### 原版 Markdown

- [x] 粗体文本
- [x] 斜体文本
- [x] 链接
- [x] 行内代码
- [x] 代码块
	- [ ] 语法高亮
- [x] 引用文本
- [ ] 图片

### LaTeX （或 KaTeX）

- [ ] 行内公式
- [ ] 公式块
- [ ] TeX 模块

### Markdown HTML 扩展

- [x] 支持插入自定义 HTML（~~基于网页本来就有~~）

### GFM

- [ ] 自定义下拉框（GFM 语法）
